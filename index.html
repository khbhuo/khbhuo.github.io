Humanoid Arm
(Solus Robots)

Solus Robots is start-up focused on building humanoid robots
for the marketing and servicing industries

At Solus Robots, I was tasked with the responsibility of designing, manufacturing, and implementing a prototype 3 axis robotic arm from beginning to end. Being a year old start-up, they had an earlier version prototype made of prefabricated components which they wanted to replace with CNCed HDPE plastic to reduce costs. I came up with a design for the arm which utilized interlocking 1/4" plastic components which provided a sturdy frame and easy to assemble design. All stuctural and rotating components of the prototype were made from this design

For this project I worked on:
SOLIDWORKS for the design
ArtCAM to make the CNC files and a CNC Router for manufacturing
Arduino and C++ for the calibration code and communications to the on board computer
Soldering and wiring electrical connections and prototype shields
Learning some EAGLE CAD to assist with the electrical design

Wireless Arduino Thermostat

Ongoing Project

I started working on this project over the summer. The plan for this is to make a thermostat which I can place in an obscure corner of my home to monitor the humidity and temperature within the house and notify me when to open the windows, turn on the AC or heat.

For this project I am using:
a DHT11 Humidity/Temperature sensor
an HC-05 Bluetooth Module
Python with PySerial and PyBluez for the PC - Arduino communication

Currently I am finishing up the script for the Bluetooth communication in Python.
The next step would be to process this data and make a GUI to display the data.

I will be updating this description with every milestone I reach in this project

Mario Kart Robot

Mechatronics Engineering Design Project

For this project we used Wii Remotes to wirelessly control NXT robots through a "Mario Kart circuit". Utilized C++ and C# for the programming. A proper engineering design process was used in designing and making the chassis of the robots.

3D Printed Wrist Joint

3D Printed forearm + hand assembly

This was a little side project I worked on during the summer. I designed the forearm to make sure that it was approximately the smae size as my forearm, and designed a universal wrist joint that was implemented to allow a wide range of motion. I used an Arduino and servos to move the hand and graphed out the range of movement using Excel.

Eventually my supervisor at Solus Robots encoporated the design into the original prototype of the robot arms!
